✅️cld 一、特休假的起源
工業革命前,農業社會工作與生活界線模糊,沒有「帶薪假」概念。19世紀工業化後,工時被嚴格制度化,勞工開始爭取「有償休息」權利。
1936年法國率先立法保障勞工每年2週帶薪假(人民陣線政府推動),被視為現代特休制度的重要起點。
1948年聯合國《世界人權宣言》第24條明確將「有薪休假」列為基本權利。
1970年國際勞工組織(ILO)第132號公約進一步規範最低天數標準(至少3週),推動各國立法。
**各國現況差異很大**
**歐洲**(尤其西歐)普遍最優渥:法國、德國、英國等多達25-30天法定特休,且通常嚴格執行「不休完不行」的文化,雇主甚至會主動要求員工休假。歐盟更規定所有會員國最低保障4週帶薪假。
**美國**則是已開發國家中的特例:**聯邦法律完全沒有強制規定帶薪特休假** 是否提供、給多少天,完全由雇主自行決定(平均約10-15天,依公司和年資而定)。這與美國強調「自由市場、雇主自主」的勞動文化有關,也常被批評是已開發國家中勞工保障最弱的一環。 **亞洲**(包含台灣、日本、韓國)制度多參考歐美架構,依年資逐年遞增天數,但實際休假率普遍偏低,與職場文化(怕影響考核、人手不足)有關,這點與歐洲「休假是義務」的氛圍形成對比。

二、人為何需要休息(上班之外)
**生理層面**
身體不是永動機,所有系統都需要修復週期:
- **肌肉與細胞修復**:活動會產生微小損傷與代謝廢物(如乳酸),休息時身體才能修復組織、清除廢物
- **大腦排毒**:睡眠時腦脊髓液會沖刷掉白天累積的代謝廢物(包含與失智相關的β類澱粉蛋白),這個機制清醒時效率很差
- **荷爾蒙重置**:壓力荷爾蒙(皮質醇)若長期偏高不下降,會干擾免疫、消化、睡眠品質,形成惡性循環
- **能量儲備**:持續輸出而不補充,會進入「透支」狀態,長期下來容易引發慢性疲勞、心血管負擔增加
**心理層面**
- **注意力資源有限**:大腦的專注力像肌肉一樣會「疲勞」,持續使用會讓判斷力、創造力、情緒控制力下降(也就是常說的「決策疲勞」)
- **情緒調節需要空間**:壓力事件需要時間「消化」,沒有留白容易累積成焦慮或耗竭(burnout)
- **記憶固化**:休息(尤其睡眠)是大腦把短期記憶轉成長期記憶的關鍵時間,熬夜或不休息會讓學習效果打折
- **身份認同的恢復**:人不是只有「工作角色」,休息讓人重新連結興趣、人際關係、自我價值感,不然容易陷入「工具化自己」的狀態
簡單說:休息不是偷懶,是身體和大腦的**必要維護程序**,跳過維護不會讓系統跑更快,只會加速故障。

三、特休假怎麼算(以台灣勞基法為例)
**天數對照(勞基法第38條)**
| 年資 | 天數 |
| 滿6個月 | 3天 |
| 滿1年 | 7天 |
| 滿2年 | 10天 |
| 滿3年 | 14天 |
| 滿5年 | 15天 |
| 滿10年 | 每多1年加1天,最高30天 |
注意:滿6個月的3天和滿1年的7天是「分開獨立計算」,不是疊加成3+7才等於滿一年的天數。
**計算制度有兩種**
1. **週年制**:從個人到職日當天起算,無論幾月幾日到職,只要工作滿6個月就有3天,滿1年就有7天,以此類推。對員工最直覺,但人資管理較麻煩(每人到職日不同)。
2. **曆年制**:企業統一以每年1月1日至12月31日為計算區間,依當年度每位員工的年資比例計算當年特休天數,所以每年1月1日會重新計算一次。對人資較好管理,但會出現「比例折算」的狀況。 例如:7月1日到職,到隔年1月1日只算「年資滿半年」,所以原本滿一年該有的7天要打對折變成3.5天,加上滿半年的3天,當年度只有6.5天,剩下3.5天會遞延到下一年度計算。

常見問題FAQ

- 特休沒休完,年度終結或離職時雇主必須折算成工資發給,不能「作廢」。
- 經勞資雙方協商,可以遞延到下一年度使用,最多只能延一次,若再次未休完就必須折現,不能無限期遞延。
- 不論公司平常用週年制或曆年制,**離職時**的特休計算一律改回週年制(依到職日、實際年資)計算,不會用曆年制的比例去打折員工權益。
- 兼職/部分工時:特休時數會依實際工時比例計算,公式是「法定特休天數 × (實際工作時數 ÷ 完整工時)」。
實務上若不確定公司用哪種制度,可以直接問人資,或用勞動部「特別休假日數試算系統」線上試算,輸入到職日就能算出正確天數。
